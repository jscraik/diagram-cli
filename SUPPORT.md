# Support

## Get help

For usage questions and bug reports, open a GitHub issue:

- [Bug report](https://github.com/jscraik/diagram-cli/issues/new?template=bug_report.md)
- [Feature request](https://github.com/jscraik/diagram-cli/issues/new?template=feature_request.md)
- [All issues](https://github.com/jscraik/diagram-cli/issues)

## What to include

Please include:

- OS and Node.js version
- Exact command run
- Expected behavior
- Actual behavior and error output
- Minimal reproduction path if possible

## Security issues

Do not report vulnerabilities in public issues.
Follow [SECURITY.md](SECURITY.md).
